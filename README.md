# BlueMoonKSP
Adds the National Team's Lander to KSP! 

Tested on KSP v1.12.5. Using any other version of KSP may cause the mod to not work correctly. 

## Dependencies (REQURIED FOR THE MOD TO WORK!)
- Module Manager

## Recommended Mods
- Waterfall
